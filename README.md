# 🎮 Корпоративные бои | Своя игра

Система для проведения корпоративной викторины в формате "Своя игра" с отборочным туром и финалом.

## 📋 Описание проекта

Проект представляет собой полноценную веб-игру для проведения корпоративных мероприятий в формате телевизионной викторины "Своя игра".

### Основные возможности:
- **Отборочный тур** - 4 вопроса, команда отвечает по очереди, каждый вопрос 10 баллов, время - 60 минут
- **Финал** - классическая "Своя игра" с таблицей 3x5 (3 категории × 5 стоимостей)
- **Система рейтинга** - топ-3 команды автоматически проходят в финал
- **Админ-панель** - управление командами, вопросами и игровыми сессиями
- **Режим отладки** - встроенная Debug-панель для ускоренного тестирования
- **Автоматическое сохранение прогресса** - можно прервать игру и продолжить позже

## 🚀 Быстрый старт

### Системные требования
- Node.js 18+ 
- MySQL 8.0+
- npm или yarn

### Установка и запуск

#### 1. Клонирование репозитория
```bash
git clone <url_репозитория>
cd corporate-quiz
```

#### 2. Установка зависимостей
```bash
# Бэкенд
cd backend
npm install

# Фронтенд (в новом терминале)
cd ../frontend
npm install
```

#### 3. Настройка базы данных

**Вариант А: Через MySQL CLI**
```bash
# Войдите в MySQL
mysql -u root -p

# Выполните команды
CREATE DATABASE corporate_quiz;
CREATE USER 'quiz_user'@'localhost' IDENTIFIED BY 'quiz123';
GRANT ALL PRIVILEGES ON corporate_quiz.* TO 'quiz_user'@'localhost';
FLUSH PRIVILEGES;
USE corporate_quiz;
SOURCE backend/database.sql;
EXIT;
```

**Вариант Б: Через любой GUI клиент (phpMyAdmin, TablePlus, DBeaver)**
1. Создайте базу данных `corporate_quiz`
2. Импортируйте файл `backend/database.sql`

#### 4. Настройка переменных окружения

**Бэкенд** (`backend/.env`):
```env
PORT=3001
DB_HOST=localhost
DB_PORT=3306
DB_USER=quiz_user
DB_PASSWORD=quiz123
DB_NAME=corporate_quiz
JWT_SECRET=super_secret_key_change_me_12345
```

**Фронтенд** (`frontend/.env`):
```env
VITE_API_URL=http://localhost:3001/api
```

#### 5. Запуск приложения

**Терминал 1 - Бэкенд:**
```bash
cd backend
npm run dev
```

**Терминал 2 - Фронтенд:**
```bash
cd frontend
npm run dev
```

#### 6. Открытие в браузере
```
http://localhost:5173
```

## 👑 Администратор

### Вход в админ-панель
1. На странице регистрации нажмите **"Админ-вход"**
2. Введите email: `admin@quiz.local`
3. Вы попадёте в админ-панель

### Что может делать администратор:

#### 1. Управление командами
- **Создание команд** - добавление команд с генерацией уникальных кодов доступа
- **Массовое создание** - добавление нескольких команд одновременно
- **Удаление команд** - очистка зарегистрированных команд
- **Сброс прогресса** - очистка прогресса квалификации для команды

#### 2. Управление вопросами
- **Отборочный тур** - загрузка вопросов из TXT/CSV/JSON файлов
- **Финальный тур** - создание категорий и вопросов через интерфейс

#### 3. Управление играми
- **Мониторинг сессий** - просмотр всех активных игровых сессий
- **Запуск игры** - одновременный старт для всех готовых команд
- **Финальные лобби** - просмотр финальных игр

#### 4. Управление видео
- **Добавление видео** - загрузка видео для заставок
- **Управление типами** - intro, between_tours, outro

#### Способ 2: Через CSV/JSON файл

3. Выберите файл

### Финальный тур

Для финала необходимо создать **3 категории** и в каждой по **5 вопросов** (всего 15 вопросов). **Без этого финал не запустится!**

#### Как добавить вопросы для финала:

1. **Перейдите в админ-панель** → раздел **"❓ Вопросы"**
2. **Выберите "Финальный"** в выпадающем списке

#### Добавление категории:
1. В левой колонке **"📂 Категории"** введите название категории
2. Нажмите **"Добавить"**

*Примеры категорий:*
- Кодировка Фано
- Количество информации
- Комбинаторика

#### Добавление вопросов в категорию:
1. Выберите категорию кликом по ней
2. В правой колонке заполните поля:
   - **Текст вопроса** - сам вопрос
   - **Правильный ответ** - точный ответ (строка)
   - **Стоимость** - 100, 200, 300, 400 или 500 баллов
3. Нажмите **"Добавить"**

#### Статус готовности финала:
В верхней части страницы отображается статус:
- ✅ **Финал готов к запуску!** - все категории и вопросы добавлены
- ❌ **Финал НЕ готов** - не хватает категорий или вопросов

**Требования для запуска финала:**
- 3 категории (ровно)
- В каждой категории по 5 вопросов (всего 15)
- Каждый вопрос имеет стоимость: 100, 200, 300, 400, 500

## 🎯 Как играть

### 1. Регистрация
- **Регистрация по коду доступа** - команды регистрируются по коду, полученному от администратора
- **Вход по коду доступа** - уже зарегистрированные команды входят по своему коду
- **Админ-вход** - вход в панель управления

### 2. Отборочный тур

**Для команды:**
1. Капитан вводит код доступа и имя капитана → регистрация
2. На главной странице нажимает **"ВОЙТИ В ИГРУ"**
3. В лобби нажимает **"ВОЙТИ В ЛОББИ"**
4. Нажимает **"ПРИГОТОВИТЬСЯ"**
5. Администратор запускает игру для всех команд

**Процесс игры:**
- 4 вопроса с вариантами ответов
- 10 баллов за правильный ответ
- 60 минут на весь тур
- Прогресс автоматически сохраняется

**После завершения:**
- Топ-3 команды автоматически становятся финалистами
- Статус финалиста отображается на главной странице

### 3. Финал

**Для финалистов:**
1. На главной странице нажимают **"ВОЙТИ В ИГРУ"**
2. Выбирают **"ТУР 2 — ФИНАЛ"**
3. Попадают в лобби финала
4. Нажимают **"Я ГОТОВ"**
5. Когда все 3 команды готовы - финал запускается автоматически

**Правила финала:**
- 3 категории по 5 вопросов (100-500 баллов)
- 6 минут на вопрос
- Команды ходят по очереди
- Можно выбирать любой доступный вопрос
- Бонус за скорость (если ответили быстрее 30 секунд)
- Побеждает команда с наибольшим количеством баллов

## 🐛 Debug режим

Для ускоренного тестирования доступна Debug-панель (кнопка в левом нижнем углу):

**Доступные функции:**
- 👑 **СТАТЬ ФИНАЛИСТОМ** - моментальный переход в финал без прохождения отбора
- 🤖 **ДОБАВИТЬ ТЕСТОВЫЕ КОМАНДЫ** - для тестирования финала (автоматически готовы)
- ⭐ **ПРОЙТИ ТУР ИДЕАЛЬНО** - автоматическое прохождение отбора с макс. баллами
- ⏭ **Пропустить вопрос** - пропуск текущего вопроса
- ⚡ **Ускорить таймер (-10 сек)** - уменьшить таймер на 10 секунд
- 🚀 **ПРИНУДИТЕЛЬНЫЙ СТАРТ ФИНАЛА** - старт финала без ожидания всех команд
- 💣 **ПОЛНОСТЬЮ ОЧИСТИТЬ БД** - очистка базы данных с сохранением админа

lsof -i :3001

kill -9 <PID>

Проверьте готовность финала в админ-панели:
1. Зайдите в раздел **"❓ Вопросы"**
2. Выберите **"Финальный"**
3. Проверьте статус - должно быть **3 категории** и **15 вопросов**

```sql
-- Проверьте количество вопросов для отбора
