
# 🎮 Корпоративные бои | Своя игра

Система для проведения корпоративной викторины в формате "Своя игра" с отборочным туром и финалом.

## 📋 Описание проекта

Проект представляет собой полноценную веб-игру для проведения корпоративных мероприятий в формате телевизионной викторины "Своя игра".

### Основные возможности:
- **Отборочный тур** - 25 вопросов, команда отвечает по очереди, каждый вопрос 10 баллов
- **Финал** - классическая "Своя игра" с таблицей 5x5 (5 категорий × 5 стоимостей)
- **Система рейтинга** - топ-3 команды автоматически проходят в финал
- **Режим отладки** - встроенная Debug-панель для ускоренного тестирования
- **Чат команды** - возможность общаться внутри команды во время игры
- **Автоматическое сохранение прогресса** - можно прервать игру и продолжить позже

## 🚀 Быстрый старт

### Системные требования
- Node.js 18+ 
- MySQL 8.0+
- npm или yarn

### Установка и запуск

#### 1. Клонирование репозитория
```bash
git clone <url_репозитория>
cd corporate-quiz
```

#### 2. Установка зависимостей
```bash
# Бэкенд
cd backend
npm install

# Фронтенд (в новом терминале)
cd ../frontend
npm install
```

#### 3. Настройка базы данных

**Вариант А: Через MySQL CLI**
```bash
# Войдите в MySQL
mysql -u root -p

# Выполните команды
CREATE DATABASE corporate_quiz;
CREATE USER 'quiz_user'@'localhost' IDENTIFIED BY 'quiz123';
GRANT ALL PRIVILEGES ON corporate_quiz.* TO 'quiz_user'@'localhost';
FLUSH PRIVILEGES;
USE corporate_quiz;
SOURCE backend/database.sql;
EXIT;
```

**Вариант Б: Через любой GUI клиент (phpMyAdmin, TablePlus, DBeaver)**
1. Создайте базу данных `corporate_quiz`
2. Импортируйте файл `backend/database.sql`

#### 4. Настройка переменных окружения

**Бэкенд** (`backend/.env`):
```env
PORT=3001
DB_HOST=localhost
DB_PORT=3306
DB_USER=quiz_user
DB_PASSWORD=quiz123
DB_NAME=corporate_quiz
JWT_SECRET=super_secret_key_change_me_12345
```

**Фронтенд** (`frontend/.env`):
```env
VITE_API_URL=http://localhost:3001/api
```

#### 5. Запуск приложения

**Терминал 1 - Бэкенд:**
```bash
cd backend
npm run dev
```

**Терминал 2 - Фронтенд:**
```bash
cd frontend
npm run dev
```

#### 6. Открытие в браузере
```
http://localhost:5173
```

## 🎯 Как играть

### 1. Регистрация
- **Формат кода**: `1-07A` (цифра-две цифры-буква)
- Пример: `1-07L` (где L - капитан), `1-07A` (сотрудник)
- Код выдан организатором мероприятия

### 2. Отборочный тур

**Для капитана:**
1. Нажмите "Создать отдел"
2. Выберите "ТУР 1 — КЛАССИЧЕСКИЙ"
3. Введите название команды
4. Поделитесь кодом команды с участниками
5. Дождитесь, пока все нажмут "Готов"
6. Нажмите "НАЧАТЬ ИГРУ"

**Для участников:**
1. Нажмите "Войти в отдел"
2. Введите код команды от капитана
3. Нажмите "Готов" в лобби
4. Ждите начала игры

**Процесс игры:**
- Вопросы задаются по очереди каждому участнику
- Капитан может отвечать за команду в любой момент
- На вопрос даётся 30 секунд
- Каждый правильный ответ приносит 10 баллов
- Всего 25 вопросов

### 3. Финал

**Квалификация в финал:**
- Топ-3 команды по итогам отборочного тура проходят в финал
- Капитан нажимает "Подтвердить выход в финал"
- Команда удаляется, статус финалиста сохраняется

**Для финалистов:**
1. Капитан нажимает "Создать отдел" → выбирает "ТУР 2 — ЭКСТРИМ / ФИНАЛ"
2. Создаёт лобби (название не важно)
3. Остальные вступают по коду лобби
4. В лобби финала капитан создаёт команду
5. Участники вступают в команду по её коду
6. Все нажимают "Я готов"
7. Капитан начинает финал

**Правила финала:**
- 5 категорий по 5 вопросов (100-500 баллов)
- 30 секунд на ответ
- Команды ходят по очереди
- Можно выбирать любой доступный вопрос
- Финальный раунд (ставки) после основных вопросов
- Побеждает команда с наибольшим количеством баллов

## 🐛 Debug режим

Для ускоренного тестирования доступна Debug-панель (кнопка в левом нижнем углу):

**Доступные функции:**
- 👑 **СТАТЬ ФИНАЛИСТОМ** - моментальный переход в финал без прохождения отбора
- 🤖 **ДОБАВИТЬ ТЕСТОВЫЕ КОМАНДЫ ДО 3-Х** - для тестирования финала
- ⭐ **ПРОЙТИ ТУР ИДЕАЛЬНО** - автоматическое прохождение отбора с макс. баллами
- ⏭ **Пропустить вопрос** - пропуск текущего вопроса
- ⚡ **Ускорить таймер (-10 сек)** - уменьшить таймер на 10 секунд
- 🎮 **ПРИНУДИТЕЛЬНЫЙ СТАРТ ФИНАЛА** - старт финала без ожидания всех команд

## 🔧 Устранение неполадок

### Ошибка подключения к базе данных
```bash
# Проверьте, запущен ли MySQL
mysql -u root -p -e "SELECT 1"

# Проверьте правильность данных в .env
cat backend/.env
```

### Порт 3001 уже используется
```bash
# Найдите процесс
lsof -i :3001

# Убейте его
kill -9 <PID>
```

### Ошибка при npm install
```bash
# Очистите кэш npm
npm cache clean --force

# Удалите node_modules и установите заново
rm -rf node_modules package-lock.json
npm install
```

### Вопросы не загружаются
```sql
-- Проверьте, есть ли вопросы в БД
SELECT COUNT(*) FROM qualification_questions;

-- Если вопросов нет, добавьте тестовые
INSERT INTO qualification_questions (category_id, difficulty_id, question_text, option_a, option_b, option_c, option_d, correct_answer) VALUES
(1, 1, 'Тестовый вопрос 1', 'А', 'Б', 'В', 'Г', 'A');
```
